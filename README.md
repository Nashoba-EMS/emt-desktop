## Setup

- Run `yarn install`
- Create `.env.production.local` file:

```
REACT_APP_API_URL="<API_URL>"
```

## Deploying

- Run `yarn deploy`
- Follow netlify directions for deploying

## Running Offline

- Start the `emt-backend`
- Run `yarn start`
